Clonage de l'appli en local

Dans un terminal rendez vous à l'endroit où vous désirez installer Bullit et lancez la commande suivante pour faire un clone du dépot Github de l'application : {% highlight bash %} $ git clone git@github.com:bullit-io/bullit.git {% endhighlight %} ou, si vous n'avez pas configuré SSH: {% highlight bash %} $ git clone https://github.com/bullit-io/bullit.git {% endhighlight %} Pour une installation locale vous pouvez aussi [télécharger Bullit en .zip](https://github.com/bullit-io/bullit/archive/master.zip)
